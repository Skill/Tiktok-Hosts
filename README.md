# Tiktok-Hosts

A list of TikTok API hosts.

These hosts have been tested for DNS resolution and timeout behaviour on a US VPS. Not all hosts are guaranteed to be accessible or stable.

## Notes

- Some hosts may be inactive, deprecated, or intermittently unavailable.
- Contributions are welcome, please open an issue if a host stops working

## Disclaimer

This repository is provided for educational and research purposes only.

The listed hosts are public-facing endpoints. This project does not promote or support misuse, abuse, or violation of any platform’s terms of service. Users are responsible for ensuring their usage complies with all applicable policies and regulations.

## Usage

You can test hosts using tools such as `curl`, `ping`, or custom benchmarking scripts to measure latency and reliability from your environment.
